# RDSSequencerCell

A compact grid cell suitable for step sequencers.

## Properties

- **active**: `bool` - Whether the cell is enabled/selected.
- **current**: `bool` - Whether the playhead is currently on this cell.
- **edit**: `bool` - Whether the cell is in edit mode. (Default: `false`)
- **cell-width**: `length` (Default: `20px`)
- **cell-height**: `length` (Default: `20px`)
- **cell-text**: `string` - Optional text label inside the cell.
- **cell-text-color**: `color` (Default: `Theme.active.text_primary`)
- **cell-text-size**: `length` (Default: `10px`)
- **cell-text-weight**: `int` (Default: `500`)

## Callbacks

- **clicked()**: Triggered when the cell is clicked.

## Example

```slint
import { RDSSequencerCell } from "rds-slint-ui-kit/components/index.slint";

RDSSequencerCell {
    active: step-enabled;
    current: root.playhead == 0;
    clicked => { step-enabled = !step-enabled; }
}
```
