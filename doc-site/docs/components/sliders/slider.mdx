# RDSSlider

A versatile slider (fader) supporting vertical and horizontal orientations.

## Properties

- **value**: `float` (Default: `0`)
- **min-value**: `float` (Default: `0`)
- **max-value**: `float` (Default: `127`)
- **size**: `string` ("small", "medium", "large") (Default: `"large"`)
- **orientation**: `string` ("vertical", "horizontal") (Default: `"vertical"`)
- **height-override**: `length` (Default: `0px`)
- **drag-sensitivity**: `float` (Default: `1.0`)
- **scroll-sensitivity**: `float` (Default: `1.0`)

## Callbacks

- **value-changed(float)**: Triggered when the value is modified.

## Example

```slint
import { RDSSlider } from "rds-slint-ui-kit/components/index.slint";

RDSSlider {
    value: 64;
    min-value: 0;
    max-value: 127;
    orientation: "vertical";
    size: "medium";
}
```
